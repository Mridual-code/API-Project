# User Directory Project

## Overview
This project fetches user data from a public API and displays it dynamically using JavaScript.

## Features
- Fetch data from API
- Display users dynamically
- Loading state
- Error handling
- Search functionality
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API

## API Used
https://jsonplaceholder.typicode.com/users

## Concepts Learned
- Async/Await
- Fetch API
- DOM Manipulation
- Event Listeners
- Array Filter Method
- Error Handling

## How to Run
1. Download or clone the project.
2. Open `index.html` in a browser.
3. Search users using the search box.
